# 🚀 KODA – Social Productivity App

KODA is a modern **social productivity app** designed to help users stay consistent, track progress, and learn together in real time.

---

## ✨ Features

### 📊 Productivity
- Track study sessions and daily goals
- Subject → Chapter → Session management
- Smart progress tracking (based on completed chapters)
- Streak system for consistency

---

### ⏱️ Focus Timer
- Pomodoro + custom timer
- Sound effects (start / pause / complete)
- Timer history tracking

---

### 📈 Analytics
- Subject-wise study insights
- Weekly progress tracking
- Performance visualization

---

### 👥 Social & Community
- Real-time activity feed
- Doubt posting and replies
- Upvote & accepted answers system
- Followers / Following system

---

### 🏆 Gamification
- Points & levels system
- Global leaderboard
- Daily leaderboard notifications (10 PM)

---

### 🔔 Real-time Updates
- WebSocket-based live updates
- No manual refresh required
- Instant notifications for:
  - Doubts
  - Replies
  - Leaderboard changes

---

## 🛠️ Tech Stack

### 📱 Mobile
- React Native (Expo)
- Zustand (State Management)
- React Navigation
- Reanimated / Moti

### 🖥️ Desktop
- React + Vite
- Electron

### ⚙️ Backend
- Node.js + Express
- MongoDB + Mongoose
- Zod (Validation)
- JWT Authentication
- Cloudinary (Image Upload)

### 🔌 Real-time
- Socket.io (WebSockets)
